# Wirecast Software — Professional Streaming Software with Camera Switching for Streamers and Broadcasters

## About Wirecast Software

Wirecast Software provides a complete professional live streaming environment with multi-camera switching for streamers, broadcasters, and television stations. Wirecast Software switches between cameras, video inputs, images, graphics, and screen captures with professional transitions. Wirecast Software supports RTMP, HLS, SRT streaming to Facebook, YouTube, Twitch, LinkedIn, and custom RTMP servers. Wirecast Software offers ISO recording, virtual camera output, instant replay, and scoreboards. Wirecast Software includes NDI support, chroma key green screen, audio mixer, and guest invitation via RTMP ingest. Wirecast Software is trusted by live streamers, esports producers, television stations, sports broadcasters, house of worship AV teams, and corporate event crews worldwide.
